# 3.4.1 Unqualified name lookup
1. The declarations from the namespace nominated by a using-directive become visible in a namespace enclosing the using-directive.
4. The declarations from the namespace nominated by the using-directive are considered members of that enclosing namespace.
3. For purposes of determining (during parsing) whether an expression is a postfix-expression for a function call, the usual name lookup rules apply.
    * postfix-expression (function): 
		```c++
		#include <iostream>
		
		void greet() {
		    std::cout << "Hello, world!" << std::endl;
		}
		
		int main() {
		    // During parsing, the compiler looks up 'greet'.
		    // Finding that it refers to a function, it treats 'greet()' as a function call.
		    greet();
		    return 0;
		}
		```
    * cast-expression (object)
		```c++
		#include <iostream>
		
		struct Functor {
		    void operator()(int x) {
		        std::cout << "Functor called with " << x << std::endl;
		    }
		};
		
		int main() {
		    Functor f;
		    // Name lookup determines that 'f' is an object of a class that overloads operator(),
		    // so 'f(42)' is parsed as a function call.
		    f(42);
		    return 0;
		}
		```
5. **[Global scope]** A name used in global scope, outside of any function, class or user-declared namespace, shall be declared before its use in global scope.
6. **[Namespace member]** A name used in a user-declared namespace outside of the definition of any function or class shall be declared
    — before its use in that namespace or
    — before its use in a namespace enclosing its namespace.
	```c++
	#include <iostream>
	
	namespace A {
	    int i = 2; // found 2nd
	    namespace N {
	         int i = 1; //  found 1st
	         int j = i;
	    }
	}
	
	int main()
	{
	    std::cout << A::N::j << std::endl;
	    return 0;
	}
	```
8. **[Namespace member function]** A name used in the definition of a function that is a member of namespace `N` (where, only for the purpose of exposition, `N` could represent the global scope) 
— shall be declared before its use in the block in which it is used or in one of its enclosing blocks or,
— shall be declared before its use in namespace `N` or,
— if `N` is a nested namespace, shall be declared before its use in one of `N`'s enclosing namespaces.
	```c++
    #include <iostream>

	int i = 5; // found 5th
	namespace A {
	    int i = 4; // found 4th
	    namespace N {
	        int i = 3; // found 3rd
	        void f();
	    }
	}
	void A::N::f() {
	    int i = 2; // found 2nd
	    {
	        int i = 1; // found 1st
	        std::cout << i << std::endl;
	    }
	}
	
	int main()
	{
	    A::N::f();
	    return 0;
	}
	```
10. **[Class member]** A name used in the definition of a class `X` outside of a member function body or nested class definition shall be declared in one of the following ways:
— before its use in class `X` (1st) or be a member of a base class of `X` (2nd), or
— if `X` is a nested class of class `Y`, before the definition of `X` in `Y` (3rd), or shall be a member of a base class of `Y` (4th, this lookup applies in turn to `Y`’s enclosing classes, starting with the innermost enclosing class),or
— if `X` is a local class or is a nested class of a local class, before the definition of class `X` in a block enclosing the definition of class `X`, or
— if `X` is a member of namespace `N`, or is a nested class of a class that is a member of `N`, or is a local class or a nested class within a local class of a function that is a member of `N`, before the definition of class `X` in namespace `N` (5th) or in one of `N`’s enclosing namespaces (6th).
    * Global class
		```c++
		#include <iostream>
		
		namespace M {
		    class B {
		         public:
		         static int i; // found 4th
		    };
		    class Z {
		         public:
		         int i = 2; // found 2nd
		    };
		}
		
		int i = 6; // found 6th
		
		namespace N {
		    int i = 5; //found 5th
		    class Y : public M::B {
		        public:
		        static int i; // found 3rd
		    	class X : public M::Z {
		            public:
		    	    int i = 1; //found 1st
		            int j = i;
		    	};
		    };
		}
		
		int N::Y::i = 3;
		int M::B::i = 4;
		
		int main()
		{
		    N::Y::X x;
		    std::cout << x.j << std::endl;
		    return 0;
		}
		```
8. **[Class member function]** A name used in the definition of a function that is a member function of class `X` shall be declared in one of the following ways:
— before its use in the block in which it is used (1st) or in an enclosing block (2nd), or
— shall be a member of class `X` (3rd) or be a member of a base class of `X` (4th), or
— if `X` is a nested class of class `Y`, shall be a member of `Y` (5th), or shall be a member of a base class of `Y` (6th, this lookup applies in turn to `Y`’s enclosing classes, starting with the innermost enclosing class), or
— if `X` is a local class or is a nested class of a local class, before the definition of class `X` in a block enclosing the definition of class `X`, or
— if `X` is a member of namespace `N`, or is a nested class of a class that is a member of `N`, or is a local class or a nested class within a local class of a function that is a member of `N`, before the member function definition, in namespace `N` (7th) or in one of `N`’s enclosing namespaces (8th).
   ```c++
   #include <iostream>
   
   class A {
        public:
        static int i; // found 6th
   
   };
   
   class B{
        public:
        int i = 4; // found 4th
   };
   
   int i = 9; // found 9th
   namespace M {
       int i = 8; // found 8th
       namespace N {
           class Y : public A {
                  public:
           	   class X : public B {
                          public:
                            void f();
                            int i = 3; // found 3rd
           	   };
                      static int i; // found 5th
           };
           int i = 7; // found 7th
           void Y::X::f() {
               int i = 2; // found 2nd
               {
                   int i = 1;// found 1st
                   std::cout << i << std::endl;
               }
           }
       }
   }
   
   int M::N::Y::i = 5;
   int A::i  = 6;
   
   int main()
   {
       M::N::Y::X x;
       x.f();
       return 0;
   }
   ```
9. Name lookup for a name used in the definition of a friend function defined inline in the class granting friendship shall proceed **as described for lookup in member function definitions**. 
— If the friend function is not defined in the class granting friendship, name lookup in the friend function definition shall proceed **as described for lookup in namespace member function definitions**.
