# Name of matrix

# Unitary matrix
Given an invertible complex square matrix $U$, if $U^{*}U=I$, where $U^{*}$ is a conjugate transpose of $U$, then $U$ is unitary.
## Properties

# Orthogonal matrix
Given an real square matrix $Q$, if $Q^{T}Q=I$, then $Q$ is orthogonal or orthonormal.
## Properties

# Hermitian matrix
Given a complex square matrix $H$, if $H^{*}=H$, where $H^{*}$ is a conjugate transpose of $H$, then $H$ is Hermitian.
