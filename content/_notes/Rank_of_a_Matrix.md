# Definition
The column rank of $A$ is the dimension of the column space of $A$, while the row rank of $A$ is the dimension of the row space of $A$.

# Properties
* $\text{rank}(A)=$ the dimension of the row space.
* $\text{rank}(A)=$ the dimension of the column space.
    * See [Proofs that column rank = row rank](https://en.wikipedia.org/wiki/Rank_(linear_algebra)#Proofs_that_column_rank_=_row_rank).
* $\text{rank}(A)=$ the number of non-zero singular values. (See [Singular-value decomposition](/sZjk4W-VQCiBPRv0axX8tg)).
* $\text{rank}(A)\leq\min\{m,n\}$. $A$ is said to have full rank if $\text{rank}(A)=\min\{m,n\}$.
* Only a zero matrix has rank zero.
* $A$ is full column rank if $\text{rank}(A)=n$.
* $A$ is full row rank if $\text{rank}(A)=m$.
